# PAC — Projetos, Atividades e Oficinas

Repositório destinado ao armazenamento das oficinas, atividades práticas, projetos e conteúdos desenvolvidos durante minha trajetória acadêmica na faculdade.

## Objetivo

Este repositório foi criado para organizar e documentar os conhecimentos adquiridos ao longo das oficinas e atividades realizadas nas disciplinas acadêmicas, servindo como:

- Histórico de aprendizado
- Portfólio acadêmico
- Material de consulta
- Registro de evolução técnica

## Estrutura do Repositório

Cada pasta representa uma oficina, atividade ou projeto específico.

Exemplo:

PAC/
│
├── Oficina-Python/
├── Oficina-C/
├── Oficina-Web/
├── Projetos/
└── README.md

## Tecnologias Utilizadas

Ao longo das oficinas e projetos, poderão ser utilizados:

- Python
- C / C++
- HTML
- CSS
- JavaScript
- Banco de Dados
- Git & GitHub
- Outras tecnologias estudadas na faculdade

## Sobre

Este repositório será atualizado constantemente conforme novas oficinas, atividades e projetos forem sendo desenvolvidos durante o curso.
